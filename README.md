# Gympass / interview-test 

## `Typescript` Implementation

### Dependencies

* NodeJs


### Before Start

```sh

npm i && npm run build

```

### Executando os testes unitários

```sh

npm run test

```
### Executando modo padrão 

```sh

npm run start

```


### Executar usando usando argumentos diferentes

```sh
node lib/index.js -l samples/racelogs.txt -c 4
```
